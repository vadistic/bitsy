# bitsy-nosql-bucket

Bitsy data store API for backendless prototyping

> bitsy
> _in British English_
> (ˈbɪtsɪ)
> ADJECTIVE
> Word forms: bitsier or bitsiest

## Idea

Simple endpoint for storing JSONs

- create bucket by posting to it (or using `/api/buckets/new`)
- add more data
- read items
